# タスクメモリの自動圧縮・要約機能 (Memory Auto-Compaction & Summarization)

## Summary
`continuous` モードなどで長期間稼働し続けるタスクにおいて、`.kage/memory/` に蓄積された過去のメモリ（JSONファイル）が一定数を超えた際、古いメモリを自律的にLLMで要約・圧縮し、コンテキストウィンドウの枯渇を防ぎつつ長期的な文脈を維持する機能の提案。

## Problem
`kage` は `.kage/memory/{task_name}/YYYY-MM-DD.json` の形式で日々の実行状態や学びを保存し、直近 N 件のメモリを次回のプロンプトに注入します。
しかし、数週間・数ヶ月にわたって稼働する長期タスク（例: 毎日のニュース要約、継続的なリファクタリング、長期調査）の場合、「直近 5 件」だけでは数週間前の重要な発見や設定方針が揮発（忘却）してしまいます。かといって読み込むメモリ数を単純に増やすと、LLM のコンテキスト上限を圧迫し、毎回の API 呼び出しコスト（入力トークン料金）が肥大化し続けます。

## Proposal
- **長期記憶の要約ファイル (`summary.json`) の導入**:
  各タスクのメモリディレクトリに、過去の要約を保持する `summary.json`（または `summary.md`）を新設します。
- **自動圧縮トリガー**:
  未要約のメモリファイルが設定値（例: `compaction_threshold: 7`）に達した場合、`kage` 内部でバックグラウンドタスクとして「古いメモリ群と現在の `summary.json` を統合し、新たな `summary.json` を生成する」専用の LLM 呼び出しを行います。
- **コンテキスト注入の最適化**:
  通常タスクの実行時は、「直近数日の生メモリ」＋「圧縮済みの `summary.json`」の両方をコンテキストとして渡すことで、トークン数を抑えつつ数ヶ月前の重要な教訓も維持できるようにします。

## Inspiration
- **MemGPT / AutoGPT の記憶管理**: 短期記憶（Short-term memory）が溢れた際に長期記憶（Long-term memory）へ要約・退避させるアーキテクチャ。
- **Claude の Context Caching**: トークン節約の仕組みですが、概念的に近い記憶の効率化。
- **Zettelkasten / Obsidian**: 日々のメモ（デイリーノート）を定期的に構造化・要約して「永遠のノート」に昇華させる運用。

## Expected Daily Benefit
- エージェントが「1週間前に試して失敗したアプローチ」を再発させる（忘却による無限ループ）のを防ぎます。
- 長期稼働する監視タスクや収集タスクのトークンコストを低く一定に保ちつつ、学習し続ける「賢いエージェント」を育てることができます。ユーザーが定期的に手動で古い JSON を間引いたり要約したりする運用保守の手間がゼロになります。

## Scope and Non-Goals
- **Scope**: メモリファイル数の監視、LLM を用いた自動要約プロンプトの実行、要約ファイルの更新、および次タスク実行時のプロンプト結合。
- **Non-Goals**: ベクトルデータベース（ChromaDB など）や RAG を用いた本格的な意味検索・知識抽出の実装はスコープ外とします。ローカル完結のファイルベース要約（フラットな階層）に留めます。

## Risks or Open Questions
- **要約による情報欠落**: LLM が圧縮時に重要なメタデータ（特定のエラーコードやパスなど）を「不要」と判断して削ってしまうリスク。要約プロンプトのチューニングが必要です。
- **圧縮タスク自体のコスト**: 要約タスクを実行するのにも API コストがかかります。安価なモデル（Gemini 2.5 Flash など）を圧縮専用に指定できるような仕組みが望ましいかもしれません。

## Suggested Rollout
1. タスクのフロントマターに `memory_compaction: true` および `compaction_model: "gemini-2.5-flash"` などの設定を追加。
2. メモリロード時に、ファイル数が閾値を超えていたら、タスク本処理の前に要約処理（Map-Reduce的なプロンプト）を挟むロジックを実装。
3. `kage memory compact <task_name>` コマンドを追加し、ユーザーが手動でも圧縮をトリガーできるようにする。
