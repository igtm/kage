# Overview (Intro)

Agent Skills は、AI エージェントに新しい能力や専門知識を提供するためのシンプルでオープンなフォーマットです。
スキルは命令、スクリプト、リソースを含むフォルダであり、エージェントがこれを見つけて使用することで、より正確かつ効率的にタスクを実行できるようになります。

## Agent Skills の利点
- **スキル作成者**: 一度構築した機能を複数のエージェント製品で展開可能。
- **互換エージェント**: ユーザーがすぐに新しい能力をエージェントに提供できる。
- **チーム・企業**: 組織の知識をポータブルでバージョン管理可能なパッケージとして保存。

## 実現可能なこと
- **ドメイン専門知識**: 法務レビューやデータ分析パイプラインなどの専門知識を再利用可能な命令としてパッケージ化。
- **新しい能力**: プレゼン作成、MCP サーバー構築、データセット分析などの機能を追加。
- **反復可能なワークフロー**: 多段階タスクを一貫性のある監査可能なワークフローに変換。
- **相互運用性**: 同じスキルを異なる互換製品で再利用可能。

## 採用状況
Claude Code, GitHub, Cursor, Mistral AI, Factory などの主要な AI 開発ツールでサポートされています。

## オープンな開発
元々は Anthropic によって開発され、現在はオープンスタンダードとして公開されており、エコシステム全体からの貢献を受け入れています。
